Manage survey for making evaluation on post types in wordpress / Gestion de formulaire pour réaliser des audit sur les post type dans Worpress

== Description ==

english

Manage survey for making evaluation on post types in Wordpress. Create several surveys and associate them to existing post types into your Wordpress installation.
You will be able to make checklist to check if all is ok on your element before publishing them

français

Gestion de formulaire pour réaliser des audit sur les post type dans Worpress. Créez plusieurs formulaires et associez chacun d'entre eux à vos types d'éléments existant dans votre installation de Wordpress.
Vous serez en mesure de créer des 'checklists' et ainsi de vérifier si tout est prêt avant de publier ces éléments

== Installation ==

english

1. Search for `wp-easy-survey` through 'Plugins -> Add' section into your wordpress admin

OR

1. Upload `wp-easy-survey` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Survey menu` for creating your first survey

français

1. Recherchez le plugin `wp-easy-survey` en passant par le menu 'Extensions -> Ajouter' dans votre administration de worpdress

Ou

1. Envoyez `wp-easy-survey` dans le dossier `/wp-content/plugins/`
2. Activer le module en passant par le menu 'Extensions' de Wordpress
3. Aller dans le menu formulaire pour créer votre premier formulaire

== Frequently Asked Questions ==

= Why my survey is not visible into elements / Pourquoi mon formulaire n'apparait pas dans les éléments de wordpress =

Check if your survey is associated to existing post type / Vérifiez si votre formulaire est bien associé à un post type (cf. screenshot-3.png)

= Why I can answer to issues / Pourquoi je ne peux répondre aux questions =

You have to "Start an evaluation" for that / Vous devez "Démarrer une évaluation"

= I made a mistake when creating an issue into a survey it is not into the good section / Je me suis trompé en créant ma question, elle n'est pas dans la bonne section =

You can drag and Drop your issues' groups | issues for ordering display / Vous pouvez glisser-déposer vos groupes de questions et vos questions pour modifier l'ordre d'affichage
